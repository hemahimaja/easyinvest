# EasyInvest 💰

**EasyInvest** is a beginner-friendly investment education web application designed for people who are new to investing, especially beginners aged 40+.  

The app explains financial concepts like **Stocks, Mutual Funds, and SIPs** in simple language and provides interactive tools such as **SIP calculators** to help users understand investment growth.

---

## 🌟 Features

• Learn Basics – Simple explanations of investment concepts  
• SIP Calculator – Calculate returns for monthly investments  
• One-Time Investment Calculator  
• Suggested Investment Plans  
• Educational YouTube learning links  
• Bilingual concept support (English & Telugu)  
• Responsive UI with smooth animations using Framer Motion

## ⚙️ Installation & Setup

### Clone the repository


git clone https://github.com/hemahimaja/easyinvest.git


---

### Run Frontend


cd easyinvest-frontend
npm install
npm start


Frontend will run on:


http://localhost:3000


---

### Run Backend


cd easyinvest-backend
npm install
node index.js


Backend will run on:


http://localhost:5000
